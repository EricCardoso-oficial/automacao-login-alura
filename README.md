# Automação na plataforma do Alura

## Descrição
Projeto de automação de login na plataforma do Alura utilizando Python(POO) e Selenium WebDriver.

## Instruções de instalação
```bash
# Clonar tudo o que estiver no repositório
git clone https://github.com/EricCardoso-oficial/automacao-login-alura.git

# Acessar a pasta do projeto
cd automacao-login-alura

# Criar o ambiente virtual
python -m venv venv

# Ativar o ambiente virtual

# Windows
venv\Scripts\activate

# Linux, macOS
source venv/bin/activate

# Instalar todas as dependências do requirements.txt
pip install -r requirements.txt
```

## Instruções de uso
Renomeie o arquivo .env-example para .env e preencha os campos correspondentes.

```bash
# Acessar pasta src
cd src

# Executar

# Windows
python main.py

# Linux, macOS
python3 main.py
```
